## -------- 亮点 --------
### 1.广泛兼容各版本系统（Windows7使用Vxkex后可用，[见此处](https://github.com/PJY1548/Break-Control-Panel/blob/main/For%20Win7.md)）
### 2.功能多样（关机重启，云盘，支持预览视频，图片，文档，小说（epub））
### 3.自定义密码保护，Hash存储，安全可靠
### 4.云盘自定义位置，管理便捷
### 5.适配移动设备，适配深色模式

## -------- 部署方式 --------
### 这里是一般性部署，Windows 7用户详见 [For Windows7](https://github.com/PJY1548/Break-Control-Panel/blob/main/For%20Win7.md) / [Windows7部署](https://github.com/PJY1548/Break-Control-Panel/blob/main/For%20Win7.md)
### 1.[安装Node.js](https://nodejs.cn/en/download)，cmd测试node npm是否正常
### 2.下载发布版本，存放至安装目录，右键setup.ps1，使用powershell运行
### 3.正常完成安装程序
### 4使用run.bat运行或直接使用node server.js
## -------- 关于 --------
### 自启动：可以自行在run.bat中添加定位到你的安装目录，使用任务计划程序实现开机自启动
### 远程使用：
#### --使用cloudflare Tunnel实现零成本云盘（当然不算电费）
